React Clothing Shop

April 2024 – May 2024

Project Overview

I worked with three friends to build a clothing shop website for a small business in Russia. I was mainly responsible for the UI design and the React frontend. The GitHub repo shows the core layout and components, but it's only a partial version: the API connection, cart, and payment logic were implemented later for the live site and were never pushed. The full site was deployed and used in production until the business stopped paying for the cloud hosting.

Note: The production version included additional features (API integration, shopping cart, payment processing) that are not present in the public repository.
